# Agent Harness Test Plan

## Scope

The harness validates agent-native access to project state, smoke checks, report export, and side-effect-free Electron Main task start planning.

## Tests

- `test_core.py`
  - Finds the repository root from the harness package.
  - Builds a structured status payload from real project files.
  - Classifies `plan.md` summary into required pending work and optional enhancement markers.
  - Builds valid smoke command definitions without running the UI.
  - Lists real Electron Main task definitions through `taskService.cjs`.
  - Builds side-effect-free dry-run task start plans with storage keys, scope ids, and payload signatures.
  - Manages project workspaces through real Electron Main `projectWorkspaceStore.cjs` against isolated temporary `userData`.
  - Exports Business Bid, AI Evaluation, and Bid Opportunity reports through real Electron Main report builders.
  - Rejects unsupported report kind / format combinations before invoking Node helpers.
- `test_full_e2e.py`
  - Invokes the installed CLI with `--json status`.
  - Invokes the installed CLI with `--json plan-summary` and verifies required work is not confused with optional enhancements.
  - Invokes the installed CLI with `--json smoke --check main-syntax` and verifies real Node syntax checks run.
  - Invokes the installed CLI with `--json list-tasks`.
  - Invokes the installed CLI with `--json start-task --type duplicate-analysis --payload-json <path> --dry-run`.
  - Invokes the installed CLI with `--json project-workspace --action list|create|set-active|get-workspace-path --user-data <temp>`.
  - Invokes the installed CLI with `--json export-report --kind rejection` and verifies a Markdown report is generated by the real Electron Main report builder.
  - Invokes the installed CLI with `--json export-report --kind rejection --format docx` and verifies a Word report is generated by the real Electron Main report builder.
  - Invokes the installed CLI with `--json export-report --kind rejection --format pdf` and verifies a PDF report is generated by the real Electron Main report builder.
  - Invokes the installed CLI with `--json export-report --kind business-bid --format xlsx` and verifies an Excel report is generated by the real Electron Main report builder.
