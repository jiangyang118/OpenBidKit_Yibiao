设置页不要做成弹窗了，也做成单独的页面。
整体分为五个部分：通用设置、文本模型配置、生图模型配置、文件解析配置、关于。

# 通用配置
当前包含显示语言、应用主题和侧边栏布局。

- 显示语言：当前固定为简体中文，不提供误导性的不可用切换。
- 应用主题：支持跟随系统、浅色、深色。Renderer 会通过根节点主题变量切换界面，Electron Main 会同步 `nativeTheme.themeSource`，让原生层不再固定浅色。
- 侧边栏布局：支持经典和紧凑；紧凑布局会默认收起侧边栏，适合小屏或需要更大工作区的场景。

新增页面时应优先使用全局 `--yb-*` 主题变量，不要在页面内硬编码白底卡片或独立持久化外观偏好。

# 文本模型配置
文本模型和之前一样，只支持OpenAI like 接口，用户输入base_url和key，可以获取可用模型，也能自己录入模型名称

# 生图模型
由于各家生图模型接口不统一，要先选择一个服务提供商，目前仅支持火山方舟和google ai studio两种方式，然后录入key，获取可用模型，也能自己录入模型名称

# 文件解析配置
文件解析是一个下拉框，包括本地解析、本地 OCR 解析、MinerU-精准解析 API、MinerU-Agent 轻量解析 API。

如果选择MinerU-精准解析 API，需要配置一个minerU的token。

配置页上显示如下对比表

| 对比维度       | 本地解析   | 本地 OCR 解析 | 🎯 精准解析 API | ⚡ Agent 轻量解析 API |
| :------------- | ---------- | ------------- | --------------- | -------------------- |
| 是否需要 Token | ✅无任何限制 | ✅无任何限制 | 需要 | 无需（IP 限频） |
| 解析速度       | ✅快 | 中等，取决于页数和 OCR 引擎 | 慢 | 中等 |
| 支持文件格式   | pdf、docx、doc、wps、txt、md | pdf、ofd、jpeg、png、bmp、webp、tif | pdf、jpeg、png、docx | pdf、jpeg、png、docx |
| 文件大小限制   | ✅无限制 | ✅无限制 | ≤ 200MB | ≤ 10MB |
| 页数限制       | ✅无限制 | ✅无限制 | ≤ 200 页 | ≤ 20 页 |
| 解析质量 | 文本型文件高 | 扫描件可用，复杂表格需人工复核 | 高 | 中 |
| 扫描件支持  | ❌不支持 | ✅支持，优先 PaddleOCR，失败回退 Tesseract | ✅支持 | 质量差 |
| 说明     | 招标文件大多数是 Word 或带文字层 PDF，本地解析可优先尝试 | 适合扫描 PDF、图片和 OFD；OFD 会先转 PDF 再 OCR，转换工具不可用时请先另存为 PDF | 复杂版面、表格还原或本地 OCR 失败时再使用 | 轻量远程解析兜底 |
